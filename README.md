## MPNetwork

[A little real-estate listing site](https://www.mpwrealestateboard.network) built on [Elixir](https://elixir-lang.org) 1.6+, [Phoenix](http://phoenixframework.org) 1.3+ and [Postgresql](https://www.postgresql.org) 9+.
